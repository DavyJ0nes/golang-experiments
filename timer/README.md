# timer

## Description

Simple CLI Countdown timer

## Usage

Usage instructions with code examples

```shell
# Install the binary
go install

# Move the alert audio file to the correct location
mkdir $HOME/.timer
cp audio/alert.mp3 $HOME/.timer/alert.mp3

# Run for 5 mins
$ timer 5m
Starting Timer for 5m0s
Timer:   00:04:54
```
